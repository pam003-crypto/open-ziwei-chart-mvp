# Open Ziwei Chart MVP

这是一个基于开源库 iztro、react-iztro、lunar-javascript 开发的紫微斗数排盘 Web App 第一版 MVP，用于学习和研究排盘功能。项目不复制任何商业软件的 UI、文案、素材或专有数据。

## 使用的开源库

- iztro：紫微斗数排盘核心算法
- react-iztro：React 紫微斗数星盘组件
- lunar-javascript：公历、农历、干支、生肖等历法辅助计算

## 安装方法

```bash
npm install
```

## 运行方法

```bash
npm run dev
```

## 构建方法

```bash
npm run build
```

## 当前功能

- 出生信息输入
- 公历 / 农历选择
- 十二时辰选择
- 紫微斗数命盘展示
- 命例保存
- 命例重新载入
- 命例删除
- JSON 导入导出
