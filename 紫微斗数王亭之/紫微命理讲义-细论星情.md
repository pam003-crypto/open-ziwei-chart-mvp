## 序论

[紫微斗数]在目前依然是一种术数。术数与学数的最大分别，是缺乏有深度的理论。

反观[紫微斗数]，虽然有[紫微斗数全书]、[紫微闻微录]等古籍，但由于编辑的工夫做得不好，而且推断之语太过欠缺理论，所以明清两代编纂类书的人，除[道藏]外，根本不将[紫微斗数]收入。

照笔者研究的结果，[紫微斗数]的精华，其实在于**统计与征验**。由统计征验，然后得出一些可靠的论断，但却仍缺乏理论基础。

本篇撰写目的，则是将笔者一己的研究心得，包括实用与理论，和盘托出。

然而必须注意，[紫微斗数]所能推断的，只是一个人的**先天运势趋向**，决定一个人的实际遭遇还有[地运]与[人运]。所谓[地运]，关系到社会背景，这一点，个人无法控制。但[人运]则不然。每个人受父母师友的影响，当临事之时，会有不同的反应。

[趋吉避凶]，其实即是控制一己的反应与决定，务求其后果较为对自己有利。因此后天人事常常可以改变先天运势。

## 上篇 · 星曜论

### 前言

[紫微斗数]的组成，分星曜与宫垣两大部分。各星曜分布于各宫垣之内，相聚相会，相离相散，即可以大致上显示出一个人的基本先天运势。

论断斗数可以分成两个步骤:

第一，先研究各宫的星曜组合，以及跟[三方四正]星曜组合的干涉，由此构成一个[星系]。

第二，再根据星系的性质，研究各宫垣干涉的情况。这时候，应该打破[三方四正]的限制。

[紫微斗数]所用，为**十四正曜、四辅曜、四佐曜、六煞曜、四化曜**，加上若干杂曜与流曜。

各星曜组合，以十四正曜为基础，共成三十八种，但分配于十二宫垣，加上各星的独坐，却可以组成一百二十星系，约一正一反而言（即如紫微在子，与紫微在午），则为六十基本星系，各有不同的特性。

## 十四正曜

### 1. 紫微（己土）

紫微为北斗主星，属**阴土**。

在斗数中，紫微至尊至贵，所以古人喻之为[帝座]。紫微不宜为[孤君]，所以最喜[百官朝拱]。

**朝拱紫微的正曜**须天府天相始为合格：
- 天府为财库、天相为印绶
- 天府最喜见禄（武曲化禄为上格，廉贞化禄次之，禄存又次之）
- 天相最喜见[财荫夹印]，最忌[刑忌夹印]

**朝拱紫微的辅曜**：左辅右弼相夹（仅丑未二宫紫微破军）、天魁天钺会合

**朝拱紫微的佐曜**：文昌文曲（增加思想优雅，抵消独断独行）、禄存天马（中和求名不求利的缺点）

**朝拱的杂曜**：三台八座（地位）、恩光天贵（名誉）、台辅封诰（声价）、龙池凤阁（灵巧）、天福天寿（通达）

**特殊情况**：

| 类型 | 表现 |
|------|------|
| [在野孤君]见空曜 | 成见甚深、思想超脱，再见华盖天刑则倾向哲理宗教 |
| [在野孤君]不见空曜 | 进退失据，不能命又不能令，却不肯低头 |
| 紫微见煞忌 | 六亲无靠（古代当僧道）或多是非纷扰 |

**紫微十二星系**：

| 宫位 | 结构 | 特点 |
|------|------|------|
| 子午 | 紫微独坐 | 午宫为日丽中天，易成良好格局 |
| 丑未 | 紫微破军 | 得左右夹则改善险阻 |
| 寅申 | 紫微天府 | 君臣庆会格 |
| 卯酉 | 紫微贪狼 | 极居卯酉格 |
| 辰戌 | 紫微天相 | 较嫌 |
| 巳亥 | 紫微七杀 | 较易成良好格局 |

### 2. 天机（乙木）

天机为南斗第一星，属**阴木**。被喻为谋士或军师。

**喜**：文昌、文曲、天才、龙池、凤阁、博士等聪明才艺之星；紫贪与天府相夹（依附权贵）；天魁天钺（一生近贵）；庙旺太阳照射的巨门（天阙）

**畏**：化忌、天虚、阴煞

**四化中的表现**：

| 化曜 | 表现 |
|------|------|
| 化权 | 最喜，表示能用于世 |
| 化科 | 聪敏且归于正道 |
| 化禄 | 经商者的聪明才智与机变 |
| 化忌见煞空 | 可能流为奸僻 |

**天机十二星系**：

| 宫位 | 结构 | 名称 |
|------|------|------|
| 子午 | 天机巨门相对 | 机巨组合 |
| 卯酉 | 天机巨门同度 | 机巨组合（卯宫最喜） |
| 辰戌 | 天机天梁同度 | 机梁组合 |
| 丑未 | 天机天梁相对 | 机梁组合 |
| 寅申 | 天机太阴同度 | 机阴组合 |
| 巳亥 | 天机太阴相对 | 机阴组合（较不喜） |

### 3. 太阳（丙火）

太阳为中天星曜，属**阳火**。日生人以太阳为中天主星。

最重要的特性是发射光和热，主**声名与贵显**。除非与财富的星曜会合（如太阴、化禄、禄存），否则主贵而不主富。

**太阳判断四原则**：
1. 先研究坐落宫位的庙旺利陷——宜庙旺不宜落陷，夜生人尤其不宜
2. 由有无财星会合判断清贵/富贵/富而不贵
3. 运限推断留意是否有开运机遇
4. 注意[中和]——过份强烈宜见收敛星曜，光热不足可靠放射性星曜

**太阳庙旺落陷**：

| 状态 | 宫位 |
|------|------|
| 庙旺 | 卯辰巳午 |
| 落陷 | 戌亥子丑（日月反背） |

**太阳化曜**：
- 化禄：主富贵
- 化权、化科：不如化禄，仅能增加贵显

**太阳十二星系**：

| 宫位 | 结构 | 名称 |
|------|------|------|
| 子午 | 太阳天梁相拱 | 阳梁组合 |
| 卯酉 | 太阳天梁同度 | 阳梁组合 |
| 丑未 | 太阳太阴同度 | 阴阳组合 |
| 辰戌 | 太阳太阴相拱 | 阴阳组合 |
| 寅申 | 太阳巨门同度 | 阳巨组合（寅宫最喜） |
| 巳亥 | 太阳巨门相拱 | 阳巨组合 |

### 4. 武曲（辛金）

武曲为北斗第六星，属**阴金**。在斗数中，武曲为财星。偏重于[质]，主行动，非常之具体。

**特性**：刚烈、孤克

**在不同宫垣的表现**：

| 宫垣 | 评价 |
|------|------|
| 财帛宫、事业宫、福德宫 | 不忌刚烈 |
| 命宫、父母宫、兄弟宫、夫妻宫、子女宫、交友宫 | 嫌孤克 |

**与煞曜的关系**：

| 组合 | 影响 |
|------|------|
| 擎羊陀罗会照 | 加强刚烈孤克 |
| 武曲化忌+羊陀相夹 | 极坏的结构 |
| 财帛宫/事业宫见羊陀 | 不忌，积极进取[武曲会羊陀，奸诈] |
| 火星铃星同度或会照 | 恶性竞争、剥削掠夺[武曲火铃同宫，因财被劫] |

**调和武曲孤克之星**：

| 星曜 | 作用 |
|------|------|
| 文昌文曲 | 柔和，但可能优柔寡断 |
| 天府 | 收敛刚烈，减少孤克 |
| 禄存/化禄 | 最喜武曲化禄，其次贪狼化禄、廉贞化禄 |

**武曲十二星系**：

| 系统 | 宫位 | 结构 |
|------|------|------|
| [武府杀] | 子午卯酉 | 武曲天府必对七杀；武曲七杀必对天府 |
| [武破相] | 寅申巳亥 | 武曲破军必对天相；武曲天相必对破军 |
| [武贪] | 辰戌丑未 | 武曲贪狼同度；武曲独坐对贪狼 |

**注意**：[武破相]一系最具刚烈孤克之性，因武曲不喜破军。

### 5. 天同（壬水）

天同为南斗第四星，属**阳水**。古人称之为[益福保生之星]。化气为福。

天同的[福]，往往经过一番挫折之后得来的安定与享受，往往象征人生的一段艰辛。

**特性**：享受（偏重物质）、软弱、浪费

**煞曜激发条件**：
- 天同带禄，不怕煞曜来侵（化禄为善，逢吉为祥）
- 天同会诸吉（昌曲），亦不怕煞曜同会

**特殊格局**：

| 格局 | 结构 | 条件 |
|------|------|------|
| [马头带剑] | 天同太阴在午+擎羊 | 丙戊年生人 |
| [反背] | 天同独坐戌宫 | 丁年生人化权，会太阴化禄、天机化科，对宫巨门化忌 |

**天同十二星系**：

| 宫位 | 结构 | 名称 |
|------|------|------|
| 子午卯酉 | 天同太阴 | 同阴组合 |
| 辰戌丑未 | 天同巨门 | 同巨组合 |
| 寅申巳亥 | 天同天梁 | 同梁组合（寅申最平稳） |

### 6. 廉贞（丁火）

廉贞为北斗第五星，属**阴火**。

具有两种不同的性格：**次桃花**（阴柔）与**化气为囚为杀**（阳刚）。

**阴阳调和关键**：

| 加强阳刚 | 加强阴柔 |
|----------|----------|
| 紫微 | 禄存 |
| 七杀 | 昌曲 |
| 破军 | 天府有禄 |
| 武曲 | 天相 |

**阳刚主贵、阴柔主富**。

**廉贞十二星系**：

| 系统 | 宫位 |
|------|------|
| [廉破相] | 子午（廉贞天相同度对破军）、卯酉（廉贞破军同度对天相） |
| [廉杀府] | 丑未（廉贞七杀同度对天府）、辰戌（廉贞天府同度对七杀） |
| [廉贪] | 寅申（廉贞与贪狼相对）、巳亥（廉贞贪狼同度） |

**[雄宿乾元格]**：
1. 廉贞七杀于未宫同度，见化禄化权化科，无煞忌
2. 廉贞在申宫独坐守命，七杀守夫妻宫于午垣又为身宫，见禄权科

### 7. 天府（戊土）

天府为南斗主星，属**阳土**。

天府所执掌的是财权，本身并不是财星，古人譬喻之为财库——即如今日之**中央银行行长**。只有运用财帛、储藏财帛的能力，而缺乏生财的能力。

**紫微与天府对比**：

| 维度 | 紫微 | 天府 |
|------|------|------|
| 地位 | 君主 | 臣佐 |
| 能力 | 制衡全局 | 偏于财权领导 |
| 开创性 | 可以开创 | 利于守成 |
| 适宜 | 开创新业 | 现成局面下从事兴革 |

**天府四库**：

| 状态 | 条件 | 表现 |
|------|------|------|
| 正常 | 有禄同会 | 财库充盈 |
| [空库] | 无禄同会、与地空天空同躔 | 巧取豪夺、外表圆融内心权术 |
| [露库] | 天相不吉、四煞刑忌交会 | 平添困扰、须设法弥缝缺陷 |

**逢府看相**：天相吉则天府权力得以发挥；天相被刑忌所夹则天府权力受制。

### 8. 太阴（癸水）

太阴属**癸水**，为中天星曜。夜生人以太阴为中天主星。

主**富**、偏重于[性]，比较抽象，显得柔和，主计划。

**庙旺落陷**：

| 状态 | 宫位 |
|------|------|
| 庙旺 | 亥子丑酉戌 |
| 落陷 | 卯辰巳午 |

**四化表现**：
- 化禄：最喜，主财源丰厚
- 化权：增加决断力
- 化科：清贵
- 化忌：财务困扰、女性问题

**喜**：禄存化禄、昌曲（文章秀发）

**特殊格局**：
- **月朗天门**：太阴在亥宫，见禄权科，为上格
- **水澄桂萼**：太阴在子，丙丁生人

### 9. 贪狼（甲木，体属水）

贪狼为北斗第一星，属**甲木**（体属水）。为**正桃花**。

**特性**：多才多艺、圆滑、交际好、物欲情欲皆重

**优点**：圆滑、玲珑、交际手腕好、才艺出众

**缺点**：沉溺嗜好、占有欲强

**喜**：火铃（**火贪格**/**铃贪格**主暴富）、昌曲（诗文风流）

**忌**：羊陀、化忌

**特殊格局**：
- **火贪格**：贪狼+火星，暴发
- **铃贪格**：贪狼+铃星，暴发
- **泛水桃花**：贪狼在亥子，见羊陀

### 10. 巨门（己土兼辛金）

巨门属**己土**（兼辛金）。为**暗曜**。

**核心象义**：口才、是非、饮食

**优点**：口才好、重信诺

**缺点**：是非口舌、多疑

**喜**：化权（增加权威性）、太阳（光明磊落，庙旺太阳化解暗曜性质）

**忌**：落陷见煞、化忌（是非口舌加剧）

---

### 11. 天相（壬水）

天相属**壬水**。为**印星**。

**核心象义**：公正、谨慎、宰相、印绶

**特性**：**逢吉则吉、逢凶则凶**，随所会合星曜而分善恶

**喜**：**财荫夹印**（化禄+天梁夹天相）

**忌**：**刑忌夹印**（化忌+擎羊夹天相）、羊陀相夹也性质不良

---

### 12. 天梁（戊土）

天梁属**戊土**。为**荫星**。

**核心象义**：长寿、解厄、荫庇、孤克

**特性**：逢凶化吉、遇难呈祥，但必须经历灾难后才化解

**优点**：有分析力、逢凶化吉

**缺点**：多灾多难、多是非、带孤克

**喜**：太阳+文昌+禄存（**阳梁昌禄格**，大利考试）

**忌**：化禄（招惹怨恨）

---

### 13. 七杀（辛金）

七杀属南斗第五星，属**阴金**。化气为权，为**将星**。

**优点**：魄力雄厚、出奇制胜、勇猛果决

**缺点**：人生寂寞、必有重大挫折、掉臂独行

**喜**：紫微天府朝拱，得贵人提拔

**忌**：落陷、化忌、四煞空劫

---

### 14. 破军（癸水）

破军为北斗第五星，属**癸水**。为**先锋、耗星**。

**特性**：去旧更新、先破坏后建设

**优点**：开创力强、领导才能

**缺点**：脾气暴躁、能攻不能守

**喜**：见禄（有后援，名为[有根]）

**忌**：空劫四煞

## 六十星系概论

以十四正曜为基础，分配于十二宫垣，约一正一反，共成**六十基本星系**。各星系配合辅佐煞化，性质变化万千。以下为部分重要星系简表：

| 星系 | 重要格局 |
|------|----------|
| 紫微在午 | 日丽中天 |
| 紫微贪狼在卯酉 | 极居卯酉 |
| 紫微天府在寅申 | 君臣庆会 |
| 天机天梁在辰戌 | 善荫朝纲 |
| 太阳在午 | 日丽中天 |
| 太阳太阴在丑未 | 日月同临 |
| 武曲天府在子午 | 将星得库 |
| 天同太阴在午+擎羊 | 马头带剑 |
| 廉贞七杀在未 | 雄宿乾元 |
| 廉贞贪狼在巳亥 | 泛水桃花 |
